# 🚖 Ride Demand Prediction

## 📌 Project Overview

This project predicts the number of rides based on date and time using a machine learning model.

---

## ⚙️ Features Used

* Hour of Day
* Day of Week
* Day
* Month
* Weekend Flag

---

## 🤖 Model

* Random Forest Regressor
* Evaluation Metric: MAE

---

## 🚀 Run the App

```bash
streamlit run app.py
```

---

## 📁 Files

* `app.py` → Streamlit web app
* `model.pkl` → Trained model
* `requirements.txt` → Dependencies
* `ride_demand.ipynb` → Model training

---

## 💡 Description

The app takes date and time as input and predicts ride demand.
